# Impeccable Design Context — EUM News

## Design Context

### Users
PE/VC 투자 전문가, 기업 IR/홍보팀, 금융·컨설팅 전문가 등 복합 사용자군.
업무 중 빠르게 뉴스를 소화하고 외부에 공유하는 **고밀도 정보 소비 워크플로우**에서 사용.
시간 압박이 있는 전문가가 주 사용자이므로, UI는 판단을 돕되 방해하지 않아야 함.

### Brand Personality
**전문적 · 신뢰 · 간결**

- 감정보다 정보를 앞세운다.
- 화려함보다 정밀함을 추구한다.
- 불필요한 요소는 제거하고 핵심만 남긴다.

### Aesthetic Direction
- **레퍼런스**: Bloomberg Terminal — 높은 정보 밀도, 절제된 색상, 기능이 곧 미학
- **테마**: 라이트·다크 모두 동등한 완성도 (`#1e3a5f` navy + `#d4af37` gold 유지)
- **안티레퍼런스**: 과도한 그라디언트, 장식적 일러스트, 소비자앱 스타일의 큰 여백
- **분위기**: 단정하고 밀도 있는 SaaS — Notion보다는 Linear, Linear보다는 Bloomberg에 가깝게

### Design Principles

1. **정보 우선 (Content-Forward)**: 크롬(UI chrome)은 최소화하고 데이터와 텍스트가 화면을 주도한다. 빈 공간은 의도적 휴식이지 채워야 할 공백이 아니다.

2. **기능이 곧 형태 (Function = Form)**: 모든 시각 요소는 기능적 이유가 있어야 한다. 장식을 위한 요소는 넣지 않는다. 버튼, 뱃지, 구분선은 모두 정보 위계를 명확히 하는 역할을 한다.

3. **신뢰를 주는 일관성 (Systematic Consistency)**: 색상(`#1e3a5f`, `#d4af37`), 반경(`rounded-xl`/`rounded-2xl`), 타이포그래피 스케일을 엄격하게 유지. 예외를 최소화해 전문적 신뢰감을 만든다.

4. **다크·라이트 동등 완성도 (Theme Parity)**: 라이트 모드를 기준으로 설계하되, 다크 모드도 단순 반전이 아닌 별도로 다듬어진 경험이어야 한다. 모든 컴포넌트에 `dark:` 변형을 완비한다.

5. **액션 명확성 (Action Clarity)**: 전문가 워크플로우에서 CTA는 즉시 식별 가능해야 한다. Primary (`#1e3a5f`), Secondary (outline), Destructive (red) 세 단계 위계를 유지한다. 금색(`#d4af37`)은 가장 중요한 단일 액션에만 사용한다.

## Tech Stack
- React 18 + TypeScript + Vite
- Tailwind CSS 4 (utility-first, no CSS variables — inline token usage)
- Lucide React (icons)
- Firebase backend

## Brand Tokens
| Token | Value | Usage |
|---|---|---|
| Navy Primary | `#1e3a5f` | Buttons, focus rings, brand anchors |
| Gold Accent | `#d4af37` | Single most-important CTA per view |
| Navy Dark | `#24456f` | Hover state for navy |
| Surface Light | `#ffffff` / `#f9fafb` | Cards and page backgrounds |
| Surface Dark | `#1f2937` / `#111827` | Dark mode cards and backgrounds |
| Text Primary | `#111827` (light) / `#f9fafb` (dark) | Body text |
| Text Muted | `#6b7280` | Secondary labels, metadata |
| Border | `#e5e7eb` (light) / `#374151` (dark) | Dividers, card borders |
| Radius Default | `rounded-xl` (0.75rem) | Standard components |
| Radius Large | `rounded-2xl` (1rem) | Panels, modals, prominent cards |
